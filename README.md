# 🐾 CalmPaws — Reactive Dog Progress Tracker

A premium mobile-first web app for tracking your reactive dog's training progress.

## Features
- 🐶 Dog profile with photo upload
- 📍 Incident logging (trigger, distance, reactivity level, technique)
- 📈 Progress charts (reactivity over time, threshold distance)
- 🔥 Calm streak tracker
- 📋 Trainer-ready PDF reports
- ⏰ Session reminders
- 💾 100% local storage — no account needed

## Tech Stack
- Vanilla HTML, CSS, JavaScript
- Chart.js for data visualization
- Google Fonts (Inter + Outfit)

## Usage
Open `index.html` in any modern browser. No installation required.
